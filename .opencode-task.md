# 任务：实现 Agent 组合推荐反馈学习功能

## 需求

用户对推荐提供有帮助/无帮助反馈（布尔值 isHelpful），系统根据反馈持续优化推荐质量。

## 实现要点

1. **反馈数据结构** (`data/recommendation-feedback.json`):
   ```json
   {
     "feedbacks": [
       { "recommendationId": "xxx", "isHelpful": true, "timestamp": 1234567890 }
     ],
     "applications": [
       { "recommendationId": "xxx", "timestamp": 1234567890 }
     ]
   }
   ```

2. **后端函数** (`lib/agent-combinations.js` 或新建 `lib/recommendation-feedback.js`):
   - `recordFeedback(recommendationId, isHelpful)` - 记录反馈
   - `recordApplication(recommendationId)` - 记录应用
   - `getFeedbackStats(recommendationId)` - 获取反馈统计
   - `getAllFeedbackStats()` - 获取所有反馈统计
   - `applyFeedbackBoost(recommendations)` - 应用反馈增益到推荐列表

3. **后端 API** (`server.js`):
   - `POST /api/agent-combinations/recommendations/:id/feedback` - 提交反馈
   - `GET /api/agent-combinations/recommendations/:id/feedback` - 获取反馈统计

4. **审计事件** (`lib/audit.js`):
   - `recommendation_feedback.submitted` - 记录反馈操作

5. **前端 UI** (`public/app.js`, `public/index.html`, `public/styles.css`):
   - 推荐卡片显示反馈按钮（👍/👎）
   - 显示反馈统计（X 人觉得有帮助）
   - 点击反馈按钮即时切换状态
   - 深色主题适配，响应式布局
